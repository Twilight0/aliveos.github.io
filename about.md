### 🚀 About Me

I'm a developer dedicated to the Python ecosystem, Linux desktop customization, and the Greek open-source community. My current focus is building **AliveOS** — a minimalist, Arch-based, developer-optimized Linux distribution.

* **🐧 Currently Building:** [**AliveOS**](https://github.com/Twilight0/aliveos) — A clean, bloat-free Linux distro powered by Arch & Garuda with a custom Cinnamon desktop.
* **🌐 AliveOS Website:** [aliveos.org](https://aliveos.org) — Landing page and project hub.
* **📁 Custom Apps:** Purpose-built applications for the AliveOS desktop experience — file manager, media player, calculator, and more.
* **🔌 Legacy Work:** Deep roots in **Kodi** add-on development ([**AliveGR**](https://github.com/Twilight0/plugin.video.alivegr)) and **ResolveURL**.
* **🌐 Localization:** Translator for **Kodi**, **ResolveURL**, and the **YouTube Addon**.
