Welcome to the official home of **AliveOS** &mdash; a minimalist, Arch-based,
developer-optimized Linux distribution. This is our first news post, so here is
a quick rundown of what we are building and what to expect in the lead-up to the
initial release.

## What is AliveOS?

AliveOS is built on a solid rolling-release base and ships a deeply customized
Cinnamon desktop, along with a small set of native utilities that follow our
zero-bloat philosophy. The goal is a clean, fast, out-of-the-way workflow that
lets you focus on building things from the first boot.

## What to expect

Between now and the `v0.1` release (target: **Q3 2026**), we will be posting
updates here as components take shape:

- **Dory File Manager** &mdash; lightweight native file navigation
- **Respite Media Player** &mdash; essential audio/video playback
- **Custom Cinnamon DE** &mdash; refined and responsive
- Pre-configured development toolchains for a ready-to-code first boot

## A quick taste

If you want to follow development, the quickest way to keep an eye on things is
to watch the repository:

```bash
git clone https://github.com/Twilight0/aliveos.github.io.git
```

> Zero Bloat Policy: AliveOS contains strictly essential utilities. It does not
> include office suites or other unnecessary pre-installed bloatware. Clean by
> default.

Thanks for stopping by &mdash; more updates soon.
