Project Manifesto & Defaults

◉
Developer-Oriented
Pre-configured development toolchains and optimized programming environments from first boot.
◉
Powered by Arch & Garuda
Solid rolling-release base with selective, optimized performance patches and kernel tweaks.
◉
Custom Cinnamon DE
A refined, deeply customized Cinnamon desktop workflow—familiar, responsive, and out of your way.
◉
Dory File Manager
A lightweight, native file manager built for essential, fast file navigation and simple operations.
◉
Respite Media Player
A streamlined audio and video player, free from heavy bloat, designed for seamless essential playback.
◉
Vivaldi Browser Default
Built for power users and speed, Vivaldi is the default choice for deep web navigation and control.
Zero Bloat Policy: This distribution contains strictly essential utilities.
It does not include office suites or other unnecessary pre-installed bloatware. Clean by default.
